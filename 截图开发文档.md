# 项目概述

开发一个 Windows 桌面截图工具，功能类似 Xnapper：按快捷键截屏后，自动生成带圆角、阴影、背景的美化图片，并支持预设和简单 AI 增强（如自动背景配色）。

目标平台：Windows（桌面）。
主要用途：内容创作者快速生成"适合社交媒体分享的精美截图"。

***

# 技术栈与架构要求

## 桌面壳

方案：Tauri（Rust + WebView）。

- 使用 Tauri 官方 JS API 和 Rust 命令实现：
  - 全局快捷键注册（调用 `tauri-plugin-global-shortcut`）。
  - 调用 Windows 系统截图能力（见截图实现章节）。

## 前端技术

- React + TypeScript + Vite。
- Tailwind CSS 做 UI 样式，风格要求：
  - 主色调：slate 系为基础色，indigo 为强调色，少量 amber 作为图标点缀（可以参考现代 SaaS UI 风格）。
  - 主要圆角：`rounded-2xl` 与 `rounded-3xl`。
  - 阴影：常规卡片使用 `shadow-sm`，主预览卡片与弹窗使用 `shadow-2xl`。
  - 有必要时使用 `backdrop-blur-sm` 制造毛玻璃效果。

## 项目结构（建议）

- `/src-tauri`
  - `src/main.rs`：Tauri 主入口，注册命令、全局快捷键和截图逻辑（Windows 实现）。
- `/src`
  - `main.tsx`：React 入口。
  - `components/`：UI 组件。
  - `modules/`：截图预览和图像处理逻辑（前端 Canvas 部分）。
  - `services/`：与 Tauri 命令交互、AI 接口调用等。
  - `types/`：类型定义（预设、配置等）。

# 核心用户流程

1. 用户按下全局快捷键（`Ctrl+Shift+Space`）。
2. 应用触发截图流程：弹出半透明遮罩窗口，用户鼠标拖选屏幕区域。
3. 截图完成后自动打开应用主窗口，展示：
   - 居中的截图预览，已经套用默认"圆角 + 阴影 + 背景"样式。
   - 右侧参数面板（圆角、阴影、背景、内边距等）。
4. 用户可以调整参数，或一键切换预设。
5. 用户点击：
   - "导出 PNG"到文件系统，或
   - "复制到剪贴板"，或
   - 直接拖拽预览图到其他应用。
6. 可选：点 "AI 配色" 按钮，让 AI 推荐背景颜色或渐变方案。

# 功能需求明细

## 1. 全局快捷键与截图

### 功能点

- 支持配置全局快捷键来启动截图流程：`Ctrl+Shift+Space`（默认）。
- 需要一个设置页面，供用户修改快捷键并保存。

### 技术要求（Tauri）

- 使用 `tauri-plugin-global-shortcut` 注册快捷键。
  - 示例：`app_handle.global_shortcut_manager().register("Ctrl+Shift+Space", || { ... })`。

### 截图实现要求（Windows）

- 实现方案：Rust 侧创建一个全屏半透明遮罩窗口（Tauri 窗口或 Win32 窗口），用户鼠标拖选区域后返回选区坐标。
- 使用 **DXGI Desktop Duplication API**（推荐）或 `BitBlt` / `PrintWindow` Win32 API 截取指定区域，返回 PNG 数据。
- 注意 **DPI 缩放处理**：Windows 高 DPI 屏幕下需将逻辑坐标转换为物理像素坐标，避免截图偏移或尺寸错误。可使用 `GetDpiForWindow` / `GetSystemDpiForProcess` 获取 DPI 缩放比例。
- 截图完成后，Tauri 将图像数据（Base64 或临时文件路径）发送给前端 React，用于后续美化处理。

## 2. 图像美化（圆角、阴影、背景）

目标：默认生成类似 Xnapper 的"带圆角、阴影、对称内边距、美观背景"的图片。

### 基础参数

每张图片至少包含以下可调参数：

- 圆角（Border radius）。
- 阴影（Shadow：强度、模糊度、偏移）。
- 内边距/边距（Padding：截图周围留白）。
- Inset / Scale（缩放截图在背景中的占比）。
- 背景类型：
  - 纯色（light/dark/自定义颜色）。
  - 渐变背景（线性/径向，高亮色+深色）。
  - 图片背景（后期可扩展）。

### 实现方式

前端 Canvas 实现，保证实时预览：

1. 创建 `<canvas>`，尺寸为最终导出尺寸（原图尺寸 + padding）。
2. 背景层：绘制纯色/渐变背景。
3. 阴影层：绘制一个比截图区域稍大的圆角矩形，使用模糊 / 带 alpha 的阴影色。
4. 内容层：
   - 画一个圆角矩形 path，`ctx.clip()`，再绘制截图（`drawImage`）。

画完后提供：

- 导出 PNG 文件（Base64 / Blob）。
- 生成可拖拽的预览（用于拖拽到其他应用）。

## 3. 预设系统（Presets）

参照 Xnapper：允许用户保存常用样式（例如"Twitter 分享"、"Blog Header"、"浅色卡片"等）。

### 预设内容

每个预设需要保存：

- 名称与 ID。
- 圆角大小。
- 阴影参数（颜色、偏移、模糊）。
- 内边距。
- 背景类型及其参数（颜色或渐变配置等）。
- 输出尺寸 / 纵横比（可选，常用社交媒体比例：Twitter/X、LinkedIn、YouTube 等）。

### 功能需求

- 预设列表展示：显示预设名和一个小缩略图（可选）。
- "应用预设"按钮：一键应用所有参数到当前截图。
- "保存为新预设"：将当前参数保存为新预设。
- 本地存储：使用 Tauri 的配置或本地 JSON 文件持久化。

## 4. 导出与分享

### 导出功能

- 导出图片为 PNG：默认 1x 尺寸，可支持 2x / 3x 缩放。
- 导出路径：默认保存到用户可配置的目录（如"美化截图"文件夹）。
- 文件命名规则：`screenshot-YYYYMMDD-HHmmss.png`。

### 快速操作

- 复制到剪贴板：一键复制当前最终图片到系统剪贴板。
- 拖拽支持：支持用户从预览区拖拽图片到其他应用（如 Notion、Slack、浏览器等）。

# AI 功能（第一阶段）

AI 不是必须，但需要预留接口。第一阶段先实现一种轻量 AI 功能：**自动背景配色**。

## 1. 自动背景配色

目标：根据截图内容，自动推荐一个"适合社交媒体分享的背景配色（纯色或渐变）"。

### 流程

1. 前端从截图图像中提取主色（颜色聚类或最常见颜色）。
2. 将提取的主色（RGB/HEX）、简单文字描述（如"深色 UI 截图"）发送给 AI 模型。
3. 模型输出：
   - 建议背景：类型（纯色/渐变）、颜色值列表（`#RRGGBB` 或 `rgba(...)`）、渐变角度等。
   - 简短用途标签（如"适合 Twitter 分享"、"适合深色模式对比"）。
4. 应用返回的背景方案到当前预览。

### AI 接口

- 预留一个 HTTP/本地函数接口：
  - `POST /ai/suggest-background`
  - 输入：主色、截图颜色分布统计、当前预设信息。
  - 输出：背景配置（类型 + 颜色列表 + 角度）。

模型可以是任意 Chat/Completion 模型（Claude / GLM / Qwen 等）。

# UI 设计需求

### 主界面布局

- 左侧：截图预览区域
  - 带背景的卡片，卡片本身圆角、阴影较强，居中显示。
- 右侧：配置面板
  - 分组：
    - "Layout"：尺寸、比例、内边距。
    - "Style"：圆角、阴影。
    - "Background"：背景类型、颜色/渐变配置、AI 配色按钮。
    - "Presets"：预设列表、保存按钮。

### 整体视觉

- 背景：浅灰（如 `bg-gray-50`），留出底部留白（例如 `pb-32`）。
- 所有弹窗（如设置窗口、导出选项）使用：
  - 遮罩：`fixed inset-0 bg-slate-900/50 backdrop-blur-sm`。
  - 容器：`bg-white rounded-3xl shadow-2xl max-w-2xl`。

# 配置与设置

需要一个"设置"或"偏好"页，至少包含：

- 全局快捷键设置：
  - 列表展示当前的快捷键。
  - 支持录制新的快捷键组合，并检测是否冲突。
- 默认导出路径。
- 默认预设选择。
- 导出选项：默认缩放比例（1x / 2x 等）、是否自动复制到剪贴板等。

# 后续扩展（非 MVP 必须）

以下功能可以作为后续版本扩展，不要求 MVP 一次实现：

- 注释功能：文本、箭头、矩形、模糊区域等，类似 Xnapper 的 Annotation。
- 自动检测截图内容边界并智能裁切，自动加对称内边距。
- OCR 提取文字，并支持 AI 总结截图内容。
- 自动隐私打码（邮箱、IP、API Key），通过 OCR + 正则或借助 AI。

# 验收标准（MVP）

一个"通过"的第一版需要满足：

1. Windows 上可以安装、运行。
2. 支持全局快捷键（`Ctrl+Shift+Space`）呼出截图，用户能框选区域并生成图片。
3. 截图会自动加上默认样式：圆角 + 阴影 + 背景 + 对称 padding。
4. 用户可以在 UI 中调整圆角、阴影、背景、padding，并实时预览。
5. 支持至少 3 个可切换预设，并支持新增/保存预设。
6. 可以导出 PNG 文件，且支持复制到剪贴板。
