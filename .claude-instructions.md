# 任务：实现 Windows 截图美化工具

请阅读 D:\Project\capture\截图开发文档.md，然后在当前目录实现这个 Windows 截图美化工具。

## 技术栈
Tauri + React + TypeScript + Vite + Tailwind CSS

## MVP 功能
1. 全局快捷键 (Ctrl+Shift+Space)
2. Windows 截图 (DXGI/BitBlt + 半透明遮罩选区)
3. Canvas 图像美化 (圆角、阴影、背景、内边距)
4. 预设系统
5. 导出功能 (PNG、剪贴板)
6. Windows DPI 缩放处理

## 要求
- 完整实现所有 MVP 功能
- 代码结构清晰
- UI 美观 (参考文档中的设计要求)
